✨ Project Features

🚦

Dynamic Signal Control

Automatically adjusts green light duration based on real-time vehicle density, reducing unnecessary waiting times.

🚑

Ambulance Priority

Detects emergency vehicles and provides an immediate green light corridor to reduce response times and save lives.

🖥️

Live Dashboard

Monitor up to four lanes simultaneously with real-time video feeds, vehicle detection boxes, and traffic light status.

📊

Data Analytics

View detailed graphs on vehicle flow, lane density, and system efficiency compared to traditional methods.

🔧 Tools & Technology

Python & Flask

🤖

YOLOv8 & OpenCV

State-of-the-art object detection using YOLOv8 and real-time video processing with OpenCV.

🌐

HTML, CSS, JS

A responsive and interactive frontend built with standard web technologies.

📈

Chart.js

Beautiful and dynamic charts for the analysis dashboard are rendered using Chart.js.
